# SharedSpace - Expense Tracker 🏠

A beautiful, minimal shared expense tracker for roommates and partners.

## Features
- **Shared Bills:** Track who paid for what and how much.
- **Auto-Split:** Automatically calculates shares and balances.
- **Settlement Tracking:** Easily record when someone pays back their share.
- **Categorization:** Organize expenses by categories like Food, Utilities, Groceries, and more.
- **Modern UI:** Clean, intuitive interface with smooth animations.

## Tech Stack
- **Flutter:** UI Framework
- **Provider:** State Management
- **Google Fonts:** Beautiful typography

## Structure
- `lib/main.dart`: App entry point and theme configuration.
- `lib/constants.dart`: Centralized palette, categories, and settings.
- `lib/models.dart`: Data structures for expenses.
- `lib/expense_state.dart`: Business logic and state management.
- `lib/dashboard_screen.dart`: Main application dashboard.
- `lib/widgets.dart`: Reusable UI components.
