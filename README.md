# 🏠 SharedSpace: Shared Expense Tracker

SharedSpace is a modern, Flutter-based mobile application designed to streamline expense tracking and bill splitting for roommates and shared living environments. Built with a core focus on **Human-Computer Interaction (HCI) principles**, it provides a frictionless experience for managing collective finances.

---

## 🚀 Key Features

- **📊 Real-time Balance Tracking**: Instantly see who owes whom with a prominent balance card and clear calls to action.
- **🏷️ Smart Categorization**: Organize expenses into logical categories like Utilities, Groceries, Internet, Food, and Transport.
- **📈 Visual Payment Split**: A dynamic progress bar visualizes the spending ratio between roommates.
- **✨ Interactive UI/UX**:
  - **Animated Transitions**: Fluid feedback for every action.
  - **Swipe-to-Delete**: Intuitive gesture-based transaction management.
  - **Bottom-Sheet Entry**: Seamlessly add new expenses without losing context.
- **💰 Local Currency Support**: Pre-configured for PKR (Rs.) with easy scalability.

---

## 🎨 HCI Design Philosophy

This project was developed with a heavy emphasis on user experience:

1. **Aesthetic & Minimalist Design**: Uses a warm "Cream & Mango" palette to reduce cognitive load and create an inviting interface.
2. **Recognition rather than Recall**: Categories use distinct icons and color-coding to help users identify transaction types at a glance.
3. **Consistency & Standards**: Adheres to Material 3 standards while maintaining a unique brand identity through custom typography (Nunito).
4. **Visibility of System Status**: Real-time updates and animated counters ensure users are always aware of the latest financial state.

## 📦 Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- A mobile emulator or physical device.

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/shared-expense-tracker.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd shared-expense-tracker
   ```
3. **Install dependencies**:
   ```bash
   flutter pub get
   ```
4. **Run the application**:
   ```bash
   flutter run
   ```
